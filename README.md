# DrivenPass
DrivenPass is an application designed for password management. Instead of remembering all your passwords, you only need to remember one, and the rest is handled by the application.

## About

Have you ever thought about how to protect yourself from online scams? Using strong and secure passwords is the best way to do it, but how can you remember random numbers and letters for every password you need to create? DrivenPass solves your problems â€“ with this app, you can safely store all your passwords, and you'll only need to remember the DrivenPass password. It's a real lifesaver.   <li>Add Cards</li>
</ul>

# How to run
  
  
</summary>
1. Clone this repository
2. Install Dependencies
```bash
npm install
```

3. Run the app with
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

4. Run the tests with
```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
