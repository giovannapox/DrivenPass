## How to run

1. Clone this repository
2. Install Dependencies
```bash
npm install
```

3. Run the app with
```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

4. Run the tests with
```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
